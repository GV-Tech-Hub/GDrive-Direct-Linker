
## How It Works
1. The tool extracts the file ID from the Google Drive sharing URL using regex
2. Constructs a direct download URL using Google Drive's download API
3. Copies the new URL to clipboard
4. Opens the download automatically

## Technical Details
- Pure HTML/CSS/JavaScript
- No external dependencies
- Client-side processing only
- Responsive design
- Dark mode optimized
